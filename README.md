# Login Campus Network without Curl

在没有curl包的情况下, 在远程服务器上登录校园网, 以位于清华大学校园网范围内的服务器为例

---

服务器在清华校内, 这是一台干净的服务器, 还没有 curl 等包, 已经 ssh 连进去了

下包配环境时发现, 需要连校园网进行 SSL 握手, 没过网络认证意味着下载不了 curl 等包, 而目前网络上关于[服务器登录校园网]()的教程大多都是基于 curl 进行校园网的登录, 故遇到了 `鸡生蛋, 蛋生鸡` 的问题

发现服务器自带Python, 故利用 `Gemini 3 Flash` 写了个 python 脚本转发校园网登录界面到端口 8888, 再利用 cursor / vscode 的端口转发功能, 将端口转发到本地笔记本的 8888 端口, 在笔记本打开界面, 输入账密登录校园网.
